# Générateur de CV

Une application web pour créer, modifier et gérer facilement des CV professionnels.

## Fonctionnalités

- **Créer un CV** : Construisez un nouveau CV à partir de zéro
- **Modifier un CV** : Modifiez les informations CV existantes
- **Exporter** : Téléchargez votre CV dans plusieurs formats
- **Modèles** : Choisissez parmi des modèles professionnels
- **Aperçu en temps réel** : Visualisez les modifications instantanément

## Installation

```bash
git clone <repository-url>
cd CV-Builder
npm install
```

## Utilisation

```bash
npm start
```

Ouvrez votre navigateur et accédez à `http://localhost:3000`

## Technologies

- HTML/CSS/JavaScript
- React (ou votre framework)
- Node.js
